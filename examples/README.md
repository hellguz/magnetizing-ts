# Examples

## Running Examples

```bash
# Build the library first
yarn build

# Run the basic usage example
npx tsx examples/basic-usage.ts
```

## Available Examples

### basic-usage.ts

Demonstrates the complete workflow:
- Defining boundaries and room requirements
- Running the discrete solver (grid-based placement)
- Running the spring solver (physics-based refinement)
- Analyzing the final layout

This is a good starting point to understand how the library works.
